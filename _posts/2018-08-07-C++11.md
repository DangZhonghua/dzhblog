### noexcept 修饰符与操作符
> 为更加简洁的处理异常， c++11 中增加了 **noexcept** 如果被noexcept 修饰的函数爆出异常，则直接terminate

```c++
noexcept keyword
int excpt_func(int ) noexcept;
void excpt_ func() noexcept(const expression);

template <class T> void fun() noexcept( noexcept( T())){};

```
### c++ 11 支持类成员的就地初始化
